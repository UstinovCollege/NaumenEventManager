# Naumen Event Manager

Веб-приложение на Flask для регистрации участников на мастер-классы с поддержкой листа ожидания, опросов, обратной связи и админ-панели.

## Возможности

- 📋 Список мастер-классов с информацией о спикере, времени, месте и количестве свободных мест
- ✅ Регистрация на мастер-класс с автоматическим листом ожидания при заполнении
- 🔔 Автоматическое продвижение из листа ожидания при отмене брони
- 🔍 Проверка статуса регистрации по контактным данным
- 💬 Форма обратной связи
- ❓ Раздел FAQ
- 📊 Опросы с подсчётом голосов и просмотром результатов
- 🔐 Админ-панель с управлением мастер-классами, регистрациями и опросами

## Стек

- **Backend:** Python 3.11, Flask
- **База данных:** SQLite
- **Frontend:** HTML + Bootstrap 5

## Запуск локально

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python app.py
```

Приложение будет доступно на `http://localhost:5000`.

## Доступ в админ-панель

- URL: `/admin`
- Пароль по умолчанию: `admin123`


## База данных

SQLite-база (`database.db`) создаётся автоматически при первом запуске и заполняется демо-данными.

Основные таблицы:

- `masterclasses` — мастер-классы
- `registrations` — регистрации участников (со статусом `confirmed` / `waitlist`)
- `polls` / `poll_options` / `poll_votes` — опросы и голоса
- `feedback` — отзывы
- `questions` — вопросы для FAQ
